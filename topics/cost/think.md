# 問い
どうしてやった方が良いことがあるのに
動画を見たりダラダラしたりするだけで
無駄な時間を過ごしてしまうのか。

# 仮説
やったっていい結果に結びつかないと思っているから
比較が以下のようになっているのではないか
1. エネルギーやコストをかけて、意味のないことをやり、ストレスを抱えるか
2. エネルギーやコストをかけず、意味のないことをやり、ストレスを避けるか

この比較なら当然、エネルギーをかけない方を選ぶに決まっている。

# しかし
しかし、エネルギーやコストをかけることが本当に避けたいことなのか。
無駄な努力を馬鹿みたいにして馬鹿にされたとして、それはどの程度のダメージなのだろうか。
漠然とした不安や恥ずかしさや危機感を元に行動を避けているのではないか。

# 本質的には
エネルギーやコストをかけることで、疲れ果ててしまったり貧困になったりする、という具体的なリスクがあるのではないか。
確かにそれは避けたい。最悪うつ病とかになると、10年単位で地獄を過ごし人生を無駄にすることになる。

本質的に、何が問題かというと、
1.  かけられるエネルギーやコストに余裕がないこと。  
    -   普通の収入の人間には、車を一台買うことは非常に大きな岐路に立たされる気分だが、  
        例えば大金持ちなら、車を一台買うことに何の躊躇いもないだろう。  
    -   逆に、普通の収入の人間なら、1000円札を投入して自販機でジュースを買うことに特に悩むことはないが、  
        金銭的に非常に余裕がない人間は、お釣りが帰って来なかったらどうしよう商品が出て来なかったらどうしようと悩むだろう。  
2.  無駄なことにコストをかけるのもまた良いという価値観を持てていないこと。
    -   自由研究のようなものに全力を傾けられる人がいる。  
        そう言った人は、単にエネルギーやコストに余裕があるだけではなく、そういう価値観を持っている。  
    -   普通大人にまでなると、リスクを取ることでリターンを得られる機会が訪れる。  
        結婚とかもそうだし、友人関係などでの提案などもそう。  
        そう言ったことを乗り越えやすい人というのは、もし無駄になってもそれほど気にしていない。
