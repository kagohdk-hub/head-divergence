# 信用取引とは
信用取引とは、買付に必要な「資金」や売却に必要な「株式」を証券会社から借りて売買する取引です。

# 制度信用取引と一般信用取引
## 制度信用取引
定義 = 取引所が公表している制度信用銘柄の選定基準を満たした銘柄のみ取引ができる信用取引のこと

デメリット = 
	取引所の選定基準を満たした銘柄しか取引ができない
	買建て・売建てから6か月以内に返済（反対売買など）をする必要がある
メリット = 
	金利が低めに設定されています。
## 一般信用取引
定義 = 投資家と証券会社の間で契約を結び、制度信用銘柄以外の銘柄についても取引が可能となる信用取引のこと

メリット = 
	幅広い銘柄で取引ができる
	返済期限が無期限であることも多い
デメリット = 
	借りた資金や株券に一定の金利上乗せがある
	→ 通常の場合、制度信用取引に比べコストが高くなります。

# 仕組み
保証金には、株券や国債、投資信託で差し入れることができるため、「気になる株があるけれど、今持っている株もまだ上がりそうで手放したくない……」という時にも、信用取引を利用することで持っている株を売らずに投資することが可能。
## 信用買い（買建て）
``` mermaid
sequenceDiagram
	participant t as トレーダー
	participant b as 信用取引口座
	participant c as 証券会社
	participant m as 株式市場

	t ->> b : 保証金
	note over t , m : 資金を借りる , 株を買う
	c ->>+ b : 資金貸し
	b ->>+ m : 買付資金
	m ->> b : 株
	note over t , m : 株を売る , 資金を返す
	b ->> m : 株
	m ->> b : 買付資金
	b ->>- c : 元金返済 + 利息
	b ->> b : 売却金額 + 利息 - 元金 の差額
	note over t , m :
	b ->> t : 保証金引き出し
```

## 信用売り（売建て）
``` mermaid
sequenceDiagram
	participant t as トレーダー
	participant b as 信用取引口座
	participant c as 証券会社
	participant m as 株式市場

	t ->> b : 保証金
	note over t , m : 株を売る , 資金を返す
	c -->> m : 貸し出し用の株を保持するための資金
	m -->> c : 貸し出し用の株を保持
	c ->>+ b : 株貸し
	b ->>+ m : 株
	m ->> b : 資金
	note over t , m : 資金を借りる , 株を買う
	b ->> m : 資金
	m ->> b : 株
	b ->>- c : 株返済 + 利息
	b ->> b : 売却金額 - 買い戻し金額 - 利息 の差額
	note over t , m :
	b ->> t : 保証金引き出し
```

# その他の知識
## 追証（おいしょう）
追証：**追**加の保**証**金
株式の変動によって、追加で担保（委託保証金）の差し入れが必要となった状況のことをいいます。

追証が発生した場合、証券会社が指定する日時までに入金などを行う必要があります。
### 追証発生のパターン1
保証金として差し入れている代用証券（株式等）は値下がりし、
委託保証金の評価額が低下し、証券会社が定める最低委託保証金維持率を下回った場合
### 追証発生のパターン2
信用の建玉が値下がり（売建ての場合値上がり）し含み損を抱えることになった場合、
その損失分が保証金から差し引かれるため、結果として保証金維持率が低下する
