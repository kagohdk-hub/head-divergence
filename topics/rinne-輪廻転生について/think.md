# 背景
## 問い
眠っている間というのは無防備になるのに、
なぜ動物は眠らないように進化しなかったのか
という問いに対して、
## 説
そもそも起きているという状態が特殊状態で
眠っている、意識がない、という状態の方が安定状態だ
という説があり、
これが最近自分の中でかなり納得度が高い。

# main
つまり、
- 生きていない (物体)
    - 生きているが意識がない (植物や眠った動物)
        - 生きていて意識がある (動物)
という関係

意識とは何か解明されていない、信号の形というだけかもしれない
そう考えると、
死ぬというのは、単により深い眠りにつくようなものと考えられるし
死ぬ前の意識と、別で生まれたものの意識が、同一のものと判定されるならば
生まれ変わりだってあり得るのではないかと思った。
